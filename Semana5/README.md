\# Semana 5 - Programadores para la Paz



\## Descripción

Servidor básico en Node.js con Express que maneja rutas GET y POST para la materia de la UdeC.



\## Rutas implementadas

\- GET / : Devuelve "Servidor activo"

\- GET /saludo : Devuelve "Hola comunidad"  

\- GET /mensaje/:nombre : Devuelve saludo personalizado

\- POST /reporte : Recibe JSON y devuelve confirmación



\## Cómo ejecutar

1\. Instalar dependencias: npm install express

2\. Ejecutar servidor: node server.js

3\. Probar en: http://localhost:3000
